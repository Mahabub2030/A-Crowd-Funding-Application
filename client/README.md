# 💰 Crowd Funding Application  

## 📌 Overview  
The **Crowd Funding Application** is a web platform that allows users to create and contribute to fundraising campaigns. It provides an easy-to-use interface for managing donation campaigns, tracking contributions, and supporting causes.  

🔗 **Live Site:** [Crowd Funding Live Site](https://crowdcube-fa18e.web.app/)  



---

## 🚀 Technologies Used  
- **Frontend:** React.js  
- **Backend:** Node.js with Express.js  
- **Database:** MongoDB  
- **Authentication:** Firebase Authentication  
- **Styling:** Tailwind CSS  
- **Deployment:** Firebase (Frontend), Heroku (Backend)  

---

## ✨ Features  
✔️ **User Authentication** – Secure login & registration via Firebase Authentication.  
✔️ **Campaign Listings** – View active donation campaigns with detailed descriptions.  
✔️ **Add Campaigns** – Users can create new fundraising campaigns.  
✔️ **Campaign Details** – View campaign descriptions, minimum donation, & deadlines.  
✔️ **Donations Tracking** – Users can track their donations & created campaigns.  
✔️ **Responsive UI** – Fully optimized for mobile & desktop using Tailwind CSS.  

---

## 📦 Dependencies  
- `react` – Frontend framework for UI.  
- `express` – Backend framework for handling APIs.  
- `mongodb` – Database for storing campaign data.  
- `firebase` – Authentication & hosting services.  
- `tailwindcss` – Utility-first CSS framework for styling.  

---

## 🔧 Installation & Setup  

Follow these steps to set up the project locally:  

### 1️⃣ Clone the repository  
```sh
git clone https://github.com/your-username/crowd-funding-app.git
cd crowd-funding-app
```

### 2️⃣ Install dependencies  
```sh
npm install
```

### 3️⃣ Configure Firebase  
- Set up Firebase Authentication in the Firebase Console.  
- Add your Firebase configuration to the `.env` file.  

### 4️⃣ Run the development server  
```sh
npm start
```
The application will be available at `http://localhost:3000/`.
